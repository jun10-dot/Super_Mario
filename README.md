# Super_Mario

## 📍 프로젝트 소개
- 고전 2D 슈퍼마리오 게임을 레퍼런스로 한 유니티 엔진 기반 Super_Mario 게임

## 🔎 프로젝트 정보
- 프로젝트 기간 : 2025년 09월 ~ 09월 (2주)
- 개발 인원 : 1명

## 🦾 기술 스택
- 언어 : C#
- 엔진 : Unity Engine
- 개발 환경 : Visual Studio

## ⌨️ 조작법
- 이동 및 점프 : (W,A,S,D) 혹은 화살표 키, SpaceBar 키
- 가속 : 키보드 Z 키 + 이동 키

## 🎯 주요 기능
- Tilemap & Grid 기반 맵 제작
- 랜덤 블록(버섯, 코인), 적, 파이프라인 상호작용 구현
- 플레이어 시스템 관리 및 가속, 점프 힘, 브레이크 제어

## 🛠️ 향후 계획
- 플레이어 상태 전환을 FSM 패턴 혹은 애니메이션 상태머신으로 구조 재설계
- 점수 및 코인 UI 추가
- 플레이어와 상호작용하는 오브젝트들은 추상 인터페이스로 구조 재조정
- 효과음 및 배경 음악
- 레벨 시스템 도입
- 메모리사용량 최적화, 결합도 낮추기
